# Installation

`@vincle/core` has zero runtime dependencies. Install it and configure
TypeScript — that's all.

## TypeScript configuration

Set `jsxImportSource` in your `tsconfig.json` so that JSX files don't need any
explicit import:

## Optional: @types/react

Installing `@types/react` gives you per-element attribute autocomplete (e.g.
`src` on `<img>`, `href` on `<a>`). It is not required — `@vincle/core` works
without it.

## What you get

### Runtime

| Export                            | Purpose                                   |
| --------------------------------- | ----------------------------------------- |
| `renderToString(node)`            | Renders a JSX tree to an HTML string      |
| `render(node)`                    | Low-level render, returns `VNode`         |
| `raw(str)`                        | Marks a trusted HTML string (no escaping) |
| `ErrorBoundary`                   | Catch rendering errors with a fallback    |
| `Fragment`                        | Standard JSX fragment (`<>…</>`)          |
| `context`, `setContext`           | Scoped context for per-request isolation  |
| `useContext`, `withScope`         | ...                                       |
| `snapshot`                        | Captures current scope for child scopes   |
| `jsx-runtime` / `jsx-dev-runtime` | JSX transform entry points (auto-wired)   |

### Types

| Export                | Purpose                                    |
| --------------------- | ------------------------------------------ |
| `VNode`               | Union of everything renderable             |
| `Component<P>`        | Type for component functions               |
| `RawString`           | Branded type for pre-escaped HTML          |
| `HTMLAttributes`      | Base HTML attribute type                   |
| `SVGAttributes`       | SVG element attribute type                 |
| `CSSProperties`       | Type for inline `style` objects            |
| `StaticAttributes<T>` | Generic to strip runtime props from a type |
| `StringEventHandlers` | String-typed event handler map             |
| `ContextKey<T>`       | Branded context key type                   |
| `ContextMap`          | Raw context map type                       |

Everything else is internal. There is no plugin registry, no provider tree, no
config object to maintain.
