## Privacy Policy

This site is a documentation site for the open-source [Vincle](https://github.com/vincle/vincle) project.

### Data collection

This site does not collect any personal data. It does not use cookies, tracking scripts, or third-party analytics.

### Local storage

The site stores your theme preference (light/dark mode) in `localStorage`. This data never leaves your browser.

### Third-party services

This site loads no third-party scripts, fonts, or resources. All assets are self-hosted.

### Contact

If you have any questions about this privacy policy, please open an issue on the [GitHub repository](https://github.com/vincle/vincle).
